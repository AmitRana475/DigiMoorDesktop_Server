Rules

This option displays all rules & regulations pertaining to Shipboard Work & Rest Hours as per maritime legistlations, applicable onboard Vessels.



This button signifies STCW Exceptions are currently Disabled, click on this button to Enable or Activate STCW Exceptions. When the button is switched to active mode, the software will start calculating STCW Exceptions from current date to all future dates. In order to calculate STCW Exceptions for previous date, either change the Computer date to previous required date then the system will calculate for the switched date to all future dates, or save the required date again, the system will then calculate for the selected crew on the selected date.


This button signifies STCW Exceptions are currently Enabled, click on this button to Disable or Deactivate STCW Exceptions. When the button is switched to deactived mode, the software will stop calculating STCW Exceptions from current date to all future dates. In order to stop calculating STCW Exceptions for previous date, either change the Computer date to previous required date then the system will calculate for the switched date to all future dates, or save the required date again, the system will then calculate for the selected crew on the selected date.

*OPA-90 regulations for a specific date can also be activated from the respective date in Work & Rest Hours Form
