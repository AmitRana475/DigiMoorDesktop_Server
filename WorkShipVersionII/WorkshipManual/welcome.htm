Connect to server

  • In server version automatically get the server pc name by work-ship, Click on "Next" button
  • In Client version it bind the list of server in dropdown list, please choose the right name and click on "Next" button

Entering Ship's Name, Flag of Registry and IMO Number:-
However, as a security layer, Work-Ship license comes with Ship's Name & IMO Number pre-fed, which cannot be changed onboard.

This information is interrogated by the Work-Ship after the User opens the Work-Ship for the first time after its successful installation. For Example:-

Ship Name: e.g. WS Ship

Flag of Vessel: e.g. BERMUDA

IMO Number: e.g. 9876543

Since IMO no is a pre-fed value, No Need of this phrase = [* Please note the IMO number is usually 7 digits long number for]

Enter Flag and click on "Next" button. To go back click on "Previous" button
