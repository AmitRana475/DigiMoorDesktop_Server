Data Backup / Restore

It is good practice to periodically back-up your Work-Ship database. This is useful in case there is a computer problem and you need to restore the program, or if you want to move Work-Ship from one computer to another.


From the MAIN MENU, select Administration

From the SUB MENU, select Backup / Restore


Clickable options as follows -


  1. Create Backup : for create database backup click on "Create Backup" button. System automatically generate backup date wise.
  2. Save as : to move or copy backup file click on "Save as" hyperlink.
  3. Delete : to delete old backup files click on "Delete" hyperlink.
  4. Restore : to Restore backup click on "Restore" hyperlink.
