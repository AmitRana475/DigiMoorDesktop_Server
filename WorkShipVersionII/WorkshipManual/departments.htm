Department

This Section allows you to Create a Department Name list.

To create a New Department name, use following steps -

  1. Click on "Add Department" button, which will open a Pop-up Window
  2. In Add Department Pop-up window, type Department Name (eg: Deck) and click on "Save" button. This will Add the new Department name to the bottom of the list.

  3. To Edit a Department Name which is already inserted*, Click on "Edit" Hyperlink, which will open a Pop-up Window with the present Department Name, edit the Name and Click on "Save" button.
  4. To Delete* a Department Name, which is already inserted*, Click on "Delete" Hyperlink, which will open a Pop-up Confirmation Window, press "Yes" button to Delete the Department Name.

* Please note Department Name assigned to any Crew member, cannot be deleted.
