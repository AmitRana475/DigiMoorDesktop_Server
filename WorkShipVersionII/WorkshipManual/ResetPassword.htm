Reset Password

This option is shown to Admin/Master & HOD with access permission. With Reset Password option, you can change or reset the password of any crew member



  1. Full Name : Type the name of the Crew member who is to be assigned as HOD. This is an auto fill field, when typed, it will suggest you available names.
  2. User Name : Information only fields, automatically filled when Full Name is inserted.
  3. New Password : Insert new password here, {Compulsory field - Password can be any alphanumeric value, minimum 6 characters}
  4. Confirm Password : Compulsory field - Confirm Password has to be same value as inserted in "Password" field

Once above details are filled, press "Reset Password" button to change or reset the password for respective Crew member.
