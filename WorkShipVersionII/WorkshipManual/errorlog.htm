Error Log

Error log feature is basically designed for the support team of the Work-Ship to investigate the error and also helps the Support team for the timely and efficient troubleshooting.


It keeps a log of the all the activities being done on the Work-Ship and it also generates a coded text file for the support team should any problems arise whilst using it.

Export this file to save & send via e-mail to Work-Ship Support team to troubleshoot any errors or as & when requested.
