What versions of Work-Ship are available?

A user can select any one of the various available versions of the Work-Ship based upon their requirements and their cost-effectiveness

1. Work-Ship ( Single User):- Basic version of Work-Ship. Best suited for 1 Crew use, accessing only 1 PC.

2. Work-Ship (Network Version):- In addition to the Single User Version this version incorporates the Server-Client relationship spectrum. This Version contains multi-user access and also consists of various additional features such as,

  • Simultaneous access from multiple computers at any given time
  • Facilitates the locking of records as to prevent against any accidental or intentional loss of entered data thereby providing an additional security layer.
  • Secure access for each user from anywhere on the Network

3. Work-Ship (Online Version) :- This version is for Shore-based management Personnel to access their Vessel/Fleet's records at any given time and also helps in building up numerous statistics and charts for their entire fleet.
