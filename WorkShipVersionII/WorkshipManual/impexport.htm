General Introduction

Administration as the name itself suggests, is for the Master and/or the System administrator of Work-Ship software. However, Admin or Master can assign a HOD for access permission. These includes numerous management related tools like backing up, importing and exporting the data, Activating and de-activating Manila Exceptions, managing Licensing, accessing & exporting Error log, accessing application log etc.


Import / Export

Work-Ship's online version incorporates the feature of Import/Export of the data. This option in Ship's version facilitates the Export of the Entered data in an database file and also helps in importing the data as received from the office. To use this feature -


From the MAIN MENU, select Administration

From the SUB MENU, select Import / Export



For Exporting the data:

  1. Select date from and to between you want to export.
  2. Click on Export button and save the this excel file*.


To Import :

  1. Click on Browse button and find the path of excel file. It will show the preview of data.
  2. Click on Import button.


*Please note that there should be no changes made to the exported/ to be imported file, if any changes are done manually, software will not accept the import/export & can result in serious database damage.
