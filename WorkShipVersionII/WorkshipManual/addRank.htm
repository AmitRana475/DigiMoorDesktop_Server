Crew Rank

This Section allows you to Create a Rank Name list.

This list can be sorted as per hierarchy of your Vessel by Simply left click + Hold & Drag Symbol button to the desired Vertical location, which makes a Rank Order sorting list, applicable to all other Sections where Crew list appears.
To create a New Rank, use following steps -

  1. Click on "Add Rank" button, which will open a Pop-up Window
  2. In Add Rank* Pop-up window, type Rank Name (eg: Chief Engineer) and click on "Save" button. This will Add the new Rank name to the bottom of the list.

  3. To Edit a Rank Name which is already inserted*, Click on "Edit" Hyperlink, which will open Pop-up Window with the present Rank Name, edit the Name and Click on "Save" button.
  4. To Delete a Rank Name*, which is already inserted*, Click on "Delete" Hyperlink, which will open a Pop-up Confirmation Window, press "Yes" button to Delete the Rank Name.


* Rank name - "Master" signifies "Captain" and is added by default, cannot be edited or deleted. In case its required to be a different Name, a new rank (eg: Captain) can be added.
In such case, it is advised, the new Rank for Master, shall be added to HOD list & given all permissions of the software use.
Please note : Rank Name assigned to any Crew member, cannot be deleted.


Managing Sorting Order of "Crew Rank" list

This list can be sorted as per hierarchy of your Vessel by Simply left click + Hold & Drag Symbol button to the desired Vertical location,
Press "Save Order" button after re-locating the positions, at last.
By Default, newly added Ranks show at the bottom of the list.
