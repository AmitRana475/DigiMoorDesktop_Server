Holiday Group

This Section allows you to Create a Holiday Group list and add Separate Holidays to allow for different Nationalities.

To create a Holiday Group name, use following steps -


  1. Click on "Add Group" button, which will open a Pop-up Window
  2. In Holiday Group Name Pop-up window, type Holiday Group Name (eg: Indian Crew 2017) and click on "Save" button. This will Add the new Holiday Group name to the bottom of the list.

  3. To Edit a Holiday Group Name which is already inserted*, Click on "Edit" Hyperlink, which will opena Pop-up Window with the present Holiday Group Name, edit the Name and Click on "Save" button.
  4. To Delete* a Holiday Group Name, which is already inserted*, Click on "Delete" Hyperlink, which will open a Pop-up Confirmation Window, press "Yes" button to Delete the Holiday Group Name.


* Please note Holiday Group Name assigned to any Crew member, cannot be deleted.

Making a Holiday List within a Holiday Group

You can insert Full Year Holidays into a Group using Holiday list, follow below steps -

  1. In the Holiday Group List, besides Delete Hyperlink of the respective Holiday Group, click on "Holiday List" hyperlink, which will open Holiday List Page
  2. On Holiday List Page, Press "Add Holiday" button, which will open a Pop-up to add Holiday Name & Date of the Holiday (calendar).
  3. Once Holiday Name & Date is filled, press on "Save" button, a message appears to add more holidays in the list, press "Yes" to add more or "No" to close the Pop-up,
  4. Press "Back" button to return to Holiday Group list page.
